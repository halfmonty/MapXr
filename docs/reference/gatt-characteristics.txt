# ─── Service 00001812 — HID (Human Interface Device) ─────────────────────────
# The TapXR presents as a composite HID device (keyboard + mouse + consumer).
# Report Map (00002A4B) defines 5 report IDs:
#   1 = Keyboard     (modifier + reserved + 6 keycodes)
#   2 = Consumer     (media keys: FastForward, Skip, Play, Stop, Vol+/-, Next, Prev)
#   3 = Relative Mouse (5 buttons + scroll wheel + tilt wheel)
#   4 = Absolute Pointer / Touchpad (12-bit X/Y)
#   5 = Extended Consumer (Play/Pause, Mute, Vol+/-, Scan Next/Prev)

00002A22  Boot Keyboard Input Report           | R N           | (current 8-byte key state)
00002A32  Boot Keyboard Output Report          | R WwR W       | (LED state: CapsLock etc.)
00002A33  Boot Mouse Input Report              | R W N         |
00002A4A  HID Information                      | R             | bcdHID=1.01 CountryCode=0 Flags=0x03 (RemoteWake|NormallyConnectable)
