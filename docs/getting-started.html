Getting Started

mapxr is a desktop application that connects to your TAP Strap wearable keyboard via Bluetooth and remaps finger-tap combinations to keyboard shortcuts, mouse actions, and custom macros.

Prerequisites

  • A TAP Strap 2 wearable keyboard
  • Windows 10+ or Linux (with BlueZ)
  • Bluetooth 4.0+ adapter

Installation

  1. Download the latest release from GitHub Releases.
  2. On Windows: run the .msi installer.
  3. On Linux: make the .AppImage executable and run it.
  4. Launch mapxr. It will appear in your system tray.

Pairing your TAP Strap

  1. Put your TAP Strap into pairing mode (hold the button until the LED flashes).
  2. Open mapxr and click Scan for devices.
  3. Select your TAP Strap from the list and click Connect.
  4. The status indicator will turn green when connected.

Loading your first profile

Profiles are JSON files that describe your tap mappings. mapxr ships with a default.json profile you can use as a starting point.

  1. Click Load Profile in the mapxr window.
  2. Browse to a .json profile file and open it.
  3. mapxr will validate the profile and activate it immediately.
See the Profiles page for the full profile JSON schema.
